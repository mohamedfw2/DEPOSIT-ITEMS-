🔥 Nouvelles Fonctionnalités

📤 Pour déposer

  1. Clique "Déposer"
  2. Sélectionne PLUSIEURS fichiers (Ctrl+Clic ou Shift+Clic)
  3. Choisis nom d'utilisateur + mot de passe
  4. Upload tous les fichiers en une fois!
  5. Partage les identifiants

📥 Pour télécharger

  1. Clique "Télécharger"
  2. Entre les identifiants
  3. Vois TOUS les fichiers
  4. Télécharge un par un ou tout en ZIP
